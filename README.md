# Weclocme to ChatHub💬
A Real-Time Chat application Built with Mern(Mongodb, Express.js, React.js and Node.js) + Socket.io. This Application allows Users to chat instantly, track online/offline and typing Status that give a seamless user chat messaging experience with a sleak and Modern Ui🎨

### Live Preview: <a href="https://amit-chathub.netlify.app">Chat-Hub</a>

## 🔖 Key Features:
 - ✅ Send & Accept Friend Requests 🤝
 - ✅ View Incoming & Outgoing Friend Requests 🔄
 - ✅ Secure Authentication Using JWT 🔐
 - ✅ Real-time Messaging with WebSockets 🌐
 - ✅ Emoji & Media Sharing 🎉📷
 - ✅ See Online & Offline Status 🟢🔴
 - ✅ Typing Indicators ✍️
 - ✅ Edit Your Profile 📝
 - ✅ Beautiful, Fully Responsive UI with Tailwind CSS 🎨
 - ✅ Smooth Animations & Dark Mode Support 🌙✨
 - ✅ Fully Responsive Design for Desktop & Mobile 📱💻

## 🛠️ Tech Stack
- 🍃 MERN(Mongodb, Express.js, React.js, Node.js)
- 🔄️ Socket.io For Real time Communication
- 🌀 Tailwind Css For A Fully Responsive UI and Cross Platform Compatability
- ☁️ Cloudinary For Image Storage
  

## Getting Started 🚀
1. Clone Repository:
```bash
git clone https://github.com/Code-By-Amit/Chat-Hub.git
```

2. Navigate to Client and Backend Folder and Install dependencies:
```bash
cd Frontend
npm install

cd Backend
npm install
```

3. Setup Environment Variables
 - For Backend
```env
PORT=3000
JWT_SECRET=
MonogDb_URL=

NODE_ENV=development
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
```

 - For Frontend
```env
VITE_BACKEND_URL=http://localhost:3000
```

4. Start The Developmenet Servers
 - For Backend
```bash
cd Backend
npm run dev
```
 - For Frontend
```bash
cd Frontend
npm run dev
```
